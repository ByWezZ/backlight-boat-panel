# Arduino

Cette partie couvre la réalisation du programme arduino utilisé pour ce panneau

## Les projets

- **ec11_rotary_encoder/** : Ce projet permet de tester le "potentiomètre" qui règle la luminosité. Je vous conseil d'aller voir la vidéo réalisée par MoThunderz : https://www.youtube.com/watch?v=fgOfSHTYeio. Le lien github: https://github.com/mo-thunderz/RotaryEncoder

- **epd2in13_V4** : Ce projet permet de tester l'afficheur e-Paper, il existe plein de version en fonction de la version de votre afficheur (voir sticker Vx sur votre afficheur, ex: V3, V4...) et des dimensions de ce dernier. Voir le github suivant pour les afficheur Waveshare : https://github.com/waveshareteam/e-Paper
